Instrucciones paso a paso

  1. Generar los archivos CSV desde el sistema de reservas Necesitarás dos archivos separados: uno para llegadas (Check-in) y otro para salidas (Check-out)
  2. Seleccionar el mes y año a visualizar El calendario solo mostrará datos para el período seleccionado
  3. Cargar ambos archivos en la herramienta Asegúrate de que:
    • El archivo de llegadas contiene la columna "Check in"
    • El archivo de salidas contiene la columna "Check-out"
  4. Generar el calendario Haz clic en "Generar Calendario" para visualizar el mapa de calor

Importante 1: Las fechas en los CSV deben estar en formato DD/MM/YYYY.

Importante 2: Los colores representan la densidad: verde para llegadas, rojo para salidas (más intenso = mayor cantidad).
